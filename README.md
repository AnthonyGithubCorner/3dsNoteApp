# 3ds Note App

A simple note app for the nintendo 3ds for non permanent note taking

# Features

- 3 colors to choose from
- Eraser
- undo and redo (right and left arrows)
- 10 pages to go through (up and down arrows)
- Clear the page (red X)

## Key Commands

| Button        | Function     |
|---------------|--------------|
| X             | Clear Screen |
| Right Trigger | Redo         |
| Left Trigger  | Undo         |
| Dpad Up       | Page Up      |
| Dpad Down     | Page Down    |
